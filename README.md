# 🏥 Rimac - Backend Citas Médicas

Aplicación backend Serverless para agendamiento médico por país (PE, CL) usando arquitectura limpia y principios SOLID.

---

## ⚙️ Tecnologías

- Node.js + TypeScript
- AWS Lambda + SQS + SNS + EventBridge
- DynamoDB + RDS (MySQL)
- Serverless Framework
- Clean Architecture
- Jest (testing)
- Swagger (OpenAPI)
